# The TeamCity Ruby Gem  [![Build Status](https://secure.travis-ci.org/jperry/teamcity-ruby-client.png?branch=master)](http://travis-ci.org/jperry/teamcity-ruby-client)

Ruby wrapper for the TeamCity Rest API

## Installation

Add this line to your application's Gemfile:

    gem 'teamcity-ruby-client'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install teamcity-ruby-client
